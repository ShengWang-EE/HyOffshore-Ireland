This code serves to evaluate the offshore wind integration in Ireland with hydrogen blending technology. It is the supporting code for the paper "Enhancing Offshore Wind Integration by Integrated Electricity and Gas Systems With Hydrogen Blending: Case of Ireland" (the title may be subject to changes). 

To run this code, other packages, including matpower 7.1, yalmip, mosek, and gurobi are required. 

This code is run from the file "main.m".

Comments are added to better understand the code.
